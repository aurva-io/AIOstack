<Step>
<StepItem title="Quick Checklist">

**Before you start, make sure you have:**

1. **Kubernetes cluster** (version 1.29+)
2. **kubectl** installed and configured
3. **Helm 3.x** installed
4. **Admin access** to your cluster


</StepItem>

<StepItem title="Kubernetes Cluster">

**Any of these work:**
- **Cloud:** GKE, EKS, AKS (version 1.29+)
- **On-premises:** Any Kubernetes distribution ( best effort support)

**Quick test:**
```bash
kubectl version
```

Expected output:
```sh
# Client Version: v1.34.1
# Kustomize Version: v5.7.1
# Server Version: v1.33.4-gke.1134000
```

</StepItem>

<StepItem title="Required Tools">

**Install kubectl:**
```bash
# macOS
brew install kubectl

# Linux
curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/linux/amd64/kubectl"
sudo install -o root -g root -m 0755 kubectl /usr/local/bin/kubectl

# Windows
choco install kubernetes-cli
```

**Install Helm:**
```bash
# macOS
brew install helm

# Linux/macOS
curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3 | bash

# Windows
choco install kubernetes-helm
```

**Verify everything works:**
```bash
kubectl version --client
helm version
```

Expected output:
```sh
kubectl version --client
# Client Version: v1.34.1
# Kustomize Version: v5.7.1
helm version
#version.BuildInfo{Version:"v3.15.2", GitCommit:"1a500d5625419a524fdae4b33de351cc4f58ec35", GitTreeState:"clean", GoVersion:"go1.22.4"}
```
</StepItem>

<StepItem title="Cluster Permissions">

**You need admin access to:**
- Create namespaces
- Deploy DaemonSets
- Create ServiceAccounts

**Test your permissions:**
```bash
kubectl auth can-i create daemonsets --all-namespaces
```

> **Success:** Should return "yes"

</StepItem>

<StepItem title="System Requirements">

**Minimum specs per node:**
- **CPU:** 1 vCPU
- **Memory:** 256MB available
- **Kernel:** Linux 5.15+ (most modern clusters have this)

**That's it!** No complex networking or special configurations needed.

</StepItem>
</Step>
